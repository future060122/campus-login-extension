# 江职大校园网自动登录

江西职业技术大学校园网自动登录 Edge扩展。

断线自动检测，秒级重连，无需手动打开登录页输入账号密码。

## 功能

- **断线检测** — 优先检测校园网关（<1 秒），不在校时自动改用互联网检测
- **自动登录** — 检测到登录页自动填写账号密码并提交
- **快速重连** — 实验性功能，每 1 秒检测网关，断线即刻重连
- **定时检测** — 可选 5~60 秒检测间隔
- **自动暂停** — 彻底离线时自动暂停，恢复后继续检测

## 安装

1. 下载本项目 ZIP 并解压
2. 打开 Edge，地址栏输入 `edge://extensions`
3. 打开左下角「开发人员模式」
4. 点击「加载解压缩的扩展」
5. 选择本项目文件夹

## 使用

1. 点击浏览器工具栏的扩展图标
2. 填入校园网账号、密码
3. 选择运营商（中国移动/联通/电信）
4. 设置检测间隔
5. 保持扩展运行即可
